Order to make this function correctly
1) Have python installed appropriately
2) Have this folder extracted to an easy to access location
3) cd to the directory of the extracted files
4) run 'pip install -r requirements.txt'
5) run 'python main.py'
6) Enjoy

General notes: Some of the examples I found had circular references for similar pages, so I only got a run of like 8 pages at once, so I suggest you populate the queue with a few unique pages to start
