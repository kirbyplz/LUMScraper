Order to make this function correctly
1) Have python installed appropriately
2) Have this folder extracted to an easy to access location
3) cd to the directory of the extracted files
4) run 'pip install -r requirements.txt'
  a) chmod a+x chromedriver if on mac
5) run 'python main.py x y'
  a) x = number of pages you want to loop
  b) y = time in seconds you want to delay between each page (say 5)
6) Enjoy
